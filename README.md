# Hotel-Management-System
A desktop-based Hotel Management System built with JavaFX, JDBC, and SQL Plus for managing rooms, guests, reservations, and billing.
##  Hotel Management System

A full-featured desktop application built using JavaFX and Java, 
connected to an SQL Plus database via JDBC.

##  Features
- Role-based login for Admin and Staff
- Room availability tracking
- Guest check-in / check-out management
- Reservation handling
- Automated invoice and billing generation
- Interactive UI built with FXML and Scene Builder

##  Tech Stack
- Java
- JavaFX / FXML / Scene Builder
- JDBC
- SQL Plus (Oracle)

##  How to Run
1. Clone the repository
2. Set up Oracle SQL Plus and create the required schema
3. Update DB credentials in the config file
4. Run the JavaFX application from Eclipse IDE
